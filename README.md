# Pscan

v1.0

use:
##### *不指定端口默认扫描常见危险端口：*<br />
##### 批量扫描(文件下需要ip.txt)<br />
```
  python3 pscan.py -p 89-90
  or
  pscan.exe -p 80-90
 ```
 ##### 单个扫描<br />
```
  python3 pscan.py -u 127.0.0.1 -p 80-90
  or
  pscan.exe -u 127.0.0.1 -p 80-90
```
